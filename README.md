### Описание проекта ###
  
Статичный одностраничный сайт. В нем освещены подходы и приемы для эффективного и плодотворного обучения в любой сфере деятельности. 

---

### Технологии в проекте ###
    
* HTML5
* CSS
* анимация объектов
* БЭМ для классов 
* организация файловой структуры по схеме nested
* iframes
* keyframes
